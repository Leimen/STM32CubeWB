STM32CubeWB Firmware Package V1.22.0 / 05-February-2025

Main Changes

Maintenance Release for BLE, 802.15.4, Thread and Zigbee updates

  • BLE updates
    • TCRL-2024-2 compliancy.
  • MAC updates
    • Improve memory management to prevent memory leaks on overloaded networks.
    • Review outdated indirect data before releasing buffers sooner.
    • Improve MAC state machine.
  • Thread updates
    • Assert added in the case of OT command called in ISR context. (FreeRTOS context only)
  • Zigbee updates
    • Fixing corner case issue related to RoutingRecord for Router and related to ZbStartupPersist and ZbStartupRejoin for EndDevice.

    • Maintenance Release for HAL/LL Drivers updates

  • HAL/LL Drivers updates
    • Update HAL/LL drivers to include latest corrections
      • Update of HAL RNG, HSEM, RTC, UART, TIM drivers.
      • Update of LL LPUART drivers.
    • Refer to release notes for further details
